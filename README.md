# Parnell_pano

A Marzipano 360 pano in Parnell. Created 31/12/2025.

## View Live

Visit the live site at: `https://sithor.github.io/Parnell_pano/`

## Deploying to GitHub Pages

This site is configured to be served from the root directory of the repository.

To deploy:
1. Commit and push all changes to the `main` branch
2. Go to your repository settings on GitHub: `https://github.com/sithor/Parnell_pano/settings/pages`
3. Under "Build and deployment":
   - Source: Select "Deploy from a branch"
   - Branch: Select `main` and `/ (root)`
   - Click Save
4. Your site will be available at `https://sithor.github.io/Parnell_pano/` within a few minutes

## Local Development

To view locally, simply open `index.html` in a web browser or use a local server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.
